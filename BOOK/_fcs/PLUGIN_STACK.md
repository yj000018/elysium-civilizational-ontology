# Plugin Stack

## Required / Recommended Plugins
- **Dataview**: For dashboards and dynamic querying of statuses, resources, and action requests.
- **Templater**: For generating nodes, action requests, and resource cards from templates.
- **QuickAdd**: For streamlining the creation process via the command palette.
- **Graph View / Local Graph**: For visualizing node connections and resource mentions.
- **Canvas**: For visual mapping and architectural planning.

## Optional Later Plugins (Watchlist)
- Pandoc Plugin
- Book Exporter
- Longform
- Draft Bench
- Quire

These optional plugins must not define FCS core behavior.
