# Dashboard Model

FCS uses Dataview-friendly dashboard documents under `BOOK/views/dashboards/`.

## STRUCTURE_DASHBOARD.md
Shows the hierarchy (Book → Parts → Chapters → Facets → Modules) with title, type, status, word count, children count, review status, missing metadata warnings, and last updated time.

## ACTION_DASHBOARD.md
Shows pending action requests, ready for AI, in progress, completed proposals, actions needing founder review, and failed / blocked actions.

## REVIEW_DASHBOARD.md
Shows modules in REVIEW_PENDING, modules needing founder review, modules with unsupported claims, modules missing sources, modules with duplicated concepts, modules exceeding target length, nodes missing summaries, and nodes lacking reader promise.

## GRAPH_RESOURCE_DASHBOARD.md
Shows content graph, concept graph, resource graph, resource index, concept index, mentions by chapter, orphan concepts, highly connected concepts, unverified resources, and public graph candidates.
