**Beats:**

**Three key concepts:**

- Spatial displacement: local improvements that export problems elsewhere

**Closing direction:** Show that displacement is a structural feature of fragmented response — and that it requires a map that can see across time, space, and domain simultaneously. Prepare OPN-006: if partial solutions displace problems, then expertise alone is not enough.
