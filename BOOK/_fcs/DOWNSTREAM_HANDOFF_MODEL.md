# Downstream Handoff Model

Handoff packages are source packages only. They are not final designed outputs.

## Handoff Types
Examples include `book_handoff`, `site_handoff`, `presentation_handoff`, and `social_handoff`.

## Contents
These packages may contain:
- clean source Markdown
- node summaries
- navigation JSON
- concept/resource indexes
- graphs
- metadata
- recommended downstream use notes

They must not contain final designed outputs (no final book design, website, deck, or social campaign).
