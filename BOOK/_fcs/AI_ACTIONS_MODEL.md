# AI Actions Model

FCS defines specific action families for AI operations.

## View
RENDER_BRIEF_VIEW, RENDER_EXPANDED_VIEW, RENDER_REVIEW_VIEW, SHOW_GRAPH_NEIGHBORHOOD, SHOW_RESOURCE_MENTIONS, EXPORT_CLEAN_MD.

## Structure
CREATE_NODE, MOVE_NODE, SPLIT_NODE, MERGE_NODES, REORDER_CHILDREN, GENERATE_INTERNAL_STRUCTURE, CONVERT_MODULE_TO_FOLDER_NODE.

## Write
WRITE_FIRST_DRAFT, CONTINUE_DRAFT, EXPAND_DRAFT, ADD_TRANSITION, GENERATE_INTRODUCTION, GENERATE_SYNTHESIS.

## Rewrite
SHORTEN, DENSIFY, CLARIFY, MAKE_MORE_NARRATIVE, MAKE_MORE_ANALYTICAL, MAKE_MORE_MANIFESTO, MAKE_MORE_ACCESSIBLE, ALIGN_TO_STYLE_GUIDE.

## Check
CHECK_CANONICAL_ALIGNMENT, CHECK_ARGUMENT_FLOW, CHECK_DUPLICATION, CHECK_MISSING_ASPECTS, CHECK_TERMINOLOGY, CHECK_SCOPE_DRIFT, CHECK_UNSUPPORTED_CLAIMS.

## Resources
EXTRACT_MENTIONED_RESOURCES, CREATE_RESOURCE_CARDS, LINK_RESOURCES_TO_MODULE, VERIFY_RESOURCE_STATUS, GENERATE_RESOURCE_INDEX, SHOW_RESOURCE_GRAPH.

## Output
RENDER_CLEAN_NODE, RENDER_EXPANDED_NODE, RENDER_MASTER_SOURCE_DOCUMENT, GENERATE_JSON_HANDOFF, GENERATE_GRAPH_HANDOFF, GENERATE_DOWNSTREAM_HANDOFF_PACKAGE.

Each action must specify: target, default agent, context policy, output policy, writeback mode, review requirement, and status after execution.
