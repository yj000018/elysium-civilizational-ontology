# Command Palette Model

This logical command palette defines the operations available in FCS. In V1, it is implemented through QuickAdd commands, Templater templates, Dataview dashboards, action request files, scripts, and Manus prompts.

## FCS Commands

### View
- Show brief
- Show expanded
- Show review view
- Show graph neighborhood
- Show resource mentions

### Structure
- Create node
- Move node
- Split node
- Merge nodes
- Reorder children
- Generate internal structure
- Convert module to folder node

### Write
- Write first draft
- Continue draft
- Expand draft
- Add transition
- Generate introduction
- Generate synthesis

### Rewrite
- Shorten
- Densify
- Clarify
- Make more narrative
- Make more analytical
- Make more manifesto
- Make more accessible
- Align to style guide

### Check
- Check canonical alignment
- Check argument flow
- Check duplication
- Check missing aspects
- Check terminology
- Check scope drift
- Check unsupported claims

### Resources
- Extract mentioned resources
- Create resource cards
- Link resources to module
- Verify resource status
- Generate resource index
- Show resource graph

### Output
- Render clean node
- Render expanded node
- Render master source document
- Generate JSON handoff
- Generate graph handoff
- Generate downstream handoff package
