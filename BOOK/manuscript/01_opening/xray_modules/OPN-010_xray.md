# Executive Summary

**EN:** This module explains why a civilizational ontology is needed: not as a theory among others, but as a map of the real dependencies between civilizational dimensions. An ontology is not an ideology — it is a map of what exists and how it is structured. The ELYSIUM architecture is this map.

**FR:** Ce module explique pourquoi une ontologie civilisationnelle est nécessaire : pas comme une théorie parmi d'autres, mais comme une carte des dépendances réelles entre les dimensions civilisationnelles. Une ontologie n'est pas une idéologie — c'est une carte de ce qui existe et de comment c'est structuré. L'architecture ELYSIUM est cette carte.
