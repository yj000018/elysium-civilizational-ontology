# Executive Summary

**EN:** This module opens the book from lived experience. Before any theory, the reader is invited to recognize a feeling they already carry: the sense that the world is not merely changing, but losing coherence. The module names this diffuse unease as civilizational — not personal, not sectoral, but systemic.

**FR:** Ce module ouvre le livre depuis l'expérience vécue. Avant toute théorie, le lecteur est invité à reconnaître un sentiment qu'il porte déjà : le sens que le monde ne change pas seulement, mais perd sa cohérence. Le module nomme ce malaise diffus comme civilisationnel — pas personnel, pas sectoriel, mais systémique.

# Binder Highlight

**FR:** Le livre commence non par une thèse mais par une reconnaissance : quelque chose se défait à une échelle pour laquelle nous n'avons pas encore de mots.
