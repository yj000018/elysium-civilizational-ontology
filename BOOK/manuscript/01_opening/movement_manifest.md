# Opening Part — Movement Manifest v0.1

---

## MOV-I — Le Malaise du Présent

| Field | Value |
|---|---|
| Movement ID | MOV-I |
| Title | Le Malaise du Présent |
| Role | Hook emotionally and diagnose the civilizational crisis without beginning in abstraction |
| Reader state at start | Vague unease; sense that something is wrong but no clear frame |
| Reader state at end | Named, understood unease; the crisis is real, systemic, and not reducible to separate problems |
| Modules | OPN-001, OPN-002, OPN-003 |
| Raw material sources | The Pathology of the Present (PROVISIONAL_MANUS_DRAFT) |
| Risks | Starting too abstractly; losing the reader before the emotional hook lands |
| Validation questions | Is the opening vignette approach right? Should OPN-001 start with a concrete scene or a named phenomenon? |

## MOV-II — L'Échec des Cartes Fragmentées

| Field | Value |
|---|---|
| Movement ID | MOV-II |
| Title | L'Échec des Cartes Fragmentées |
| Role | Show why existing mental maps, institutions, disciplines, and expert domains cannot solve a meta-crisis alone |
| Reader state at start | Understands the crisis is systemic; still believes existing tools might be sufficient |
| Reader state at end | Sees why fragmented expertise and siloed institutions structurally cannot address the meta-crisis |
| Modules | OPN-004, OPN-005, OPN-006 |
| Raw material sources | The Illusion of Silos (PROVISIONAL_MANUS_DRAFT) |
| Risks | Appearing to dismiss expertise; alienating specialist readers |
| Validation questions | Should OPN-006 explicitly honor expertise before critiquing its limits? Is 3 modules sufficient for this movement? |
